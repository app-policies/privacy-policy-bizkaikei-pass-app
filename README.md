# ビジネス会計Pass プライバシーポリシー

最終更新日: 2026年6月26日

ビジネス会計Pass（以下「本アプリ」）は、ビジネス会計検定試験2級・3級の学習機能、広告表示、利用状況の解析、アプリ内課金を提供するために、端末上の学習履歴および外部サービスが提供する情報を利用する場合があります。

本アプリは、ログインやアカウント作成を必要としません。

## 収集する情報

本アプリは、以下の情報を取得または利用する場合があります。

- 学習履歴、回答履歴、正誤履歴、苦手問題、連続学習日数など、アプリ内で保存される学習データ
- アプリの設定情報
- 広告配信や広告効果測定に必要な端末情報、広告表示に関する情報
- アプリの改善に必要な利用状況の集計値
- アプリ内課金の購入状態を管理するために必要な識別子および購入情報
- クラッシュ、不具合、利用状況の把握に必要な技術情報

本アプリは、氏名、住所、電話番号、連絡先、正確な位置情報を、アカウント情報として独自に取得しません。

本アプリは、利用者が個別企業の財務諸表、経営情報、投資判断、税務相談、会計相談、経営相談に関する個別情報を入力・保存・送信する機能を持ちません。

## 利用目的

取得または利用する情報は、以下の目的で使用します。

- 問題演習、復習、学習進捗表示の提供
- 誤答優先学習、苦手問題、連続学習日数などの学習機能の提供
- 広告表示および広告効果測定
- 利用状況の集計、品質改善
- アプリ内課金の購入状態の確認、復元、管理
- 不具合調査、品質改善、クラッシュ解析
- お問い合わせへの対応

## 学習データの保存

学習履歴、回答履歴、苦手問題、連続学習日数などの学習データは、原則として利用者の端末内に保存されます。

本アプリは、学習履歴を管理するための独自アカウントサーバーを運営していません。端末の削除、アプリの削除、端末変更などにより、端末内の学習データが失われる場合があります。

## 広告について

本アプリでは、Google Mobile Ads を利用して、無料版でバナー広告、インタースティシャル広告、リワード広告を表示する場合があります。プレミアム利用中は広告を表示しません。

Google Mobile Ads では、広告配信、広告効果測定、不正防止などのために、端末情報や広告表示に関する情報が利用される場合があります。これらの情報の取り扱いは、Google のポリシーに従って管理されます。

本アプリは、IDFAを利用せず、App Tracking Transparency の許可を要求しません。

## 利用状況の解析について

本アプリでは、Firebase Analytics を利用して、アプリ改善のために利用状況の集計値を取得します。

送信する情報は、アプリ起動、学習開始、回答、学習完了、購入導線などの画面・導線名と、回答数、正答数、正解率、対象級、分野などの集計値に限定します。

本アプリは、Firebase Analytics に次の情報を送信しません。

- 問題本文、選択肢本文、解説本文
- 問題IDの生値
- 氏名、メールアドレス、自由入力、その他の個人情報
- 購入取引ID、広告ID、SDKキー、アプリ内部ID

## アプリ内課金について

本アプリでは、非消耗型のアプリ内課金により、ビジネス会計検定試験2級・3級対策の学習機能を拡張する場合があります。

購入処理は Apple の仕組みを通じて行われ、購入状態の管理には RevenueCat を利用します。RevenueCat では、購入状態の確認、復元、課金機能の提供に必要な識別子や購入情報が取り扱われる場合があります。

## 第三者サービス

本アプリでは、広告配信、課金管理、品質改善のために、以下の第三者サービスを利用する場合があります。各サービスで取り扱われる情報は、それぞれのプライバシーポリシーに従って管理されます。

- Google Mobile Ads
- Firebase Analytics
- RevenueCat
- Apple が提供する App Store、アプリ内課金、クラッシュログ、診断関連サービス

## 外部送信

広告配信、課金管理、利用状況の集計、クラッシュ解析、品質改善のため、第三者サービスに端末情報、広告表示に関する情報、購入状態の確認に必要な情報、利用状況の集計値、クラッシュ情報、診断情報などが送信される場合があります。

本アプリは、利用者が入力した氏名、住所、電話番号、連絡先、正確な位置情報、個別企業の財務諸表、経営情報、投資判断、税務相談、会計相談、経営相談に関する個別情報を独自に収集して外部送信する機能を持ちません。

また、問題本文、選択肢本文、解説本文を利用状況解析のために外部送信しません。
